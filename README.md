# THE BRIDGE: Command Centre

The centralized operating system for Shane & Lori. 

## Objectives
- **Visibility:** Real-time "What's Happened / What's Next" for all projects.
- **Coordination:** Integrated calendars and task management.
- **Velocity:** Reducing friction for the Captain and providing analytical clarity for Lori.

## Tech Stack (Planned)
- **Frontend:** Next.js (Hosted on Vercel)
- **Data Source:** GitHub (Markdown Roadmaps & Agent Logs)
- **Integrations:** Google/Microsoft Calendar, High Peek Pro (CRM)
